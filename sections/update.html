🔁 Actualización del CLI Mantenimiento

💡
Idea clave: Claude Code se actualiza frecuentemente — nuevas features, correcciones de bugs, cambios en comandos. Mantenerlo actualizado evita comportamientos inesperados y da acceso a las últimas capacidades.

🔍 Verificar la versión instalada

claude --version

O dentro de una sesión:

/doctor

/doctor muestra la versión instalada y puede indicar si hay una actualización disponible.

🔄 Actualizar según el método de instalación

Método de instalaciónCómo actualizarNotas
Instalador nativo (PowerShell) Automático en segundo plano El instalador nativo gestiona sus propias actualizaciones. Si querés forzar una actualización, volvé a correr el script de instalación.
WinGet winget upgrade Anthropic.ClaudeCode Requiere ejecución manual. No se actualiza automáticamente.
npm npm update -g @anthropic-ai/claude-code Actualiza a la última versión en npm. Verificá con claude --version después.

⚠️ Qué puede cambiar entre versiones

ComponenteEstabilidadQué hacer
Formato de settings.jsonGeneralmente estable, cambios menores entre versiones mayoresRevisá claude config list después de actualizar si algo deja de funcionar
Skills y CLAUDE.mdMuy estable — formato Markdown + frontmatter YAMLNo suelen romperse entre versiones
Hooks (nombres de eventos)Estable para los eventos principales (PreToolUse, PostToolUse, Stop)Verificá con /hooks que siguen activos después de actualizar
Slash commands built-inPueden agregarse o modificarseUsá /help para ver los disponibles en la versión actual
MCP (protocolo)Sigue la spec pública — generalmente retrocompatibleActualizá también los paquetes de servidores MCP periódicamente

📦 Actualizar servidores MCP instalados globalmente

Si instalaste servidores MCP vía npm global, actualizalos periódicamente:

# Actualizar todos los paquetes globales de npm
npm update -g

# Actualizar un servidor específico
npm update -g @modelcontextprotocol/server-github

🔧 Si algo deja de funcionar después de actualizar

  1. Ejecutá /doctor — reporta problemas de configuración o incompatibilidades detectadas
  2. Verificá claude config list — comparás la configuración activa con lo que esperás
  3. Si hay hooks o skills que dejaron de funcionar, reiniciá la sesión (/exit + claude) y verificá con /skills y /hooks
  4. En caso de error grave, desinstalar y reinstalar con el instalador oficial suele resolver problemas de estado corrupto
# Reinstalar limpio (instalador nativo)
irm https://claude.ai/install.ps1 | iex

# Reinstalar limpio (npm)
npm uninstall -g @anthropic-ai/claude-code
npm install -g @anthropic-ai/claude-code
📌
Las actualizaciones de Claude Code no afectan tus archivos de configuración (settings.json, CLAUDE.md, skills, commands). Solo reemplazan el binario.
