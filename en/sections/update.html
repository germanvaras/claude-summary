🔁 CLI Updates Maintenance↗ Docs

💡
Key concept: Claude Code updates frequently — new features, bug fixes, command changes. Keeping it updated avoids unexpected behavior and gives access to the latest capabilities.

🔍 Check the installed version

claude --version

Or from within a session:

/doctor

/doctor shows the installed version and may indicate if an update is available.

🔄 Update by installation method

Installation methodHow to updateNotes
Native installer (PowerShell) Automatic in the background The native installer manages its own updates. To force an update, run the installation script again.
WinGet winget upgrade Anthropic.ClaudeCode Requires manual execution. Does not update automatically.
npm npm update -g @anthropic-ai/claude-code Updates to the latest version on npm. Verify with claude --version afterwards.

⚠️ What can change between versions

ComponentStabilityWhat to do
settings.json formatGenerally stable, minor changes between major versionsCheck claude config list after updating if something stops working
Skills and CLAUDE.mdVery stable — Markdown + YAML frontmatter formatRarely break between versions
Hooks (event names)Stable for main events (PreToolUse, PostToolUse, Stop)Verify with /hooks that they are still active after updating
Built-in slash commandsMay be added or modifiedUse /help to see those available in the current version
MCP (protocol)Follows the public spec — generally backward compatibleAlso update MCP server packages periodically

📦 Update globally installed MCP servers

If you installed MCP servers via global npm, update them periodically:

# Update all global npm packages
npm update -g

# Update a specific server
npm update -g @modelcontextprotocol/server-github

🔧 If something stops working after updating

  1. Run /doctor — it reports configuration issues or detected incompatibilities
  2. Check claude config list — compare the active configuration with what you expect
  3. If hooks or skills stopped working, restart the session (/exit + claude) and verify with /skills and /hooks
  4. For a serious error, uninstalling and reinstalling with the official installer usually fixes corrupted state issues
# Clean reinstall (native installer)
irm https://claude.ai/install.ps1 | iex

# Clean reinstall (npm)
npm uninstall -g @anthropic-ai/claude-code
npm install -g @anthropic-ai/claude-code
📌
Claude Code updates do not affect your configuration files (settings.json, CLAUDE.md, skills, commands). They only replace the binary.
